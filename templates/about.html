About Text To QR

A compact Flask app to turn text or URLs into downloadable QR codes.

What is this app?

Text To QR is a lightweight web application built with Flask that generates a QR code from any text or URL you provide. The QR image is streamed directly as a PNG file—no files are saved on the server.

Key Features

  • Convert text/URLs into QR codes dynamically
  • Instant PNG download with proper cache control
  • In-memory processing (no file storage)
  • Environment-driven configuration
  • Clean, minimal UI with Bootstrap

How it works

On the home page, enter your text or URL and submit the form. The server validates the input (max 1000 characters), generates a QR code using the qrcode library, and returns a qrcode.png file for download.

Dependencies

  • Flask – Web framework
  • qrcode – QR code generation
  • python-dotenv – Environment variable management

Configuration

  • IS_DEBUG (str): Set to true/1 to enable Flask debug mode
  • FLASK_RUN_HOST (str): Host to bind (default 0.0.0.0)
  • FLASK_RUN_PORT (int): Port to run on (default 5000)

Tip: Inputs are not stored; they're processed in-memory and immediately discarded.
